How this page stays honest

Every completed bout gets logged to data/reconciliation.jsonl via scripts/reconcile.mjs within hours of the event ending. The log captures the prediction snapshot (Advantage Score, Predictability, confidence tier) as of the last pre-fight odds refresh, plus the actual winner/method/round from authoritative sources, plus CLV (opening line vs closing line movement toward our pick).

The numbers above are computed fresh on every site build from that append-only log. We don't edit past records. If the model is wrong, the loss shows up here immediately.

Compare these live numbers to our backtest accuracy (59.8% on 325 historical bouts). If the two diverge materially — live accuracy significantly below backtest — we'd recalibrate weights.
